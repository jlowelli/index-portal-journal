# index-portal-journal

一个用于归档和发布独立 HTML 页面的仓库。这里存放的页面内容彼此独立，不针对特定域名或网站，主要用于记录、展示或测试各类前端页面资源。

## 项目简介

本仓库旨在收集和整理多个独立的 HTML 文件，每个文件均可作为完整页面单独运行。这些页面可能包含演示、笔记、工具、静态原型或其他前端相关内容，适合用于快速查阅、分享或存档。

## 目录说明

```
.
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面共用资源（如样式、脚本、图片等）
├── README.md       # 本文件
└── index.html      # 可选：页面索引入口
```

- **pages/**：每个 HTML 文件为一个独立页面，文件名应简洁明了，便于识别。
- **assets/**：存放多个页面可能共用的静态资源，如图片、CSS、JavaScript 等。
- **index.html**：可选的入口页面，用于列出所有可访问的页面链接，方便浏览。

## 页面归档说明

- 每个页面应保持独立，尽量避免跨页面依赖。
- 页面文件推荐使用语义化命名，例如 `tool-calendar.html`、`demo-animation.html` 等。
- 如需引用共用资源，请使用相对路径指向 `assets/` 目录。
- 页面内容允许包含说明文字、注释或作者信息，但不应包含任何具体域名、网址或营销性内容。

## 维护说明

- 欢迎通过 Pull Request 提交新的独立页面或改进现有页面。
- 提交前请确保页面在主流浏览器中正常显示，代码整洁可读。
- 请勿添加与页面内容无关的广告、跟踪代码或恶意脚本。
- 本仓库保持开放与中立，不鼓励任何形式的商业推广或诱导点击行为。
